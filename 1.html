== :- It is used for comparison between two variables. It will check value of the varisble. === :- It is used for stict comparison between two variables. It will check the type and value of both variables. (...) It allows to quickly copy all or part of an existing array or object into another array or object. ex:- const numberOne = [1,2,3]; const numberTwo = [4,5,6]; const numberCombined = [...numbersOne, ...numberTwo]; var:- The scope of a var variable is functional scope. It can be updated and re-declared into the scope. It can be declared without initialization. It cant have variables with same name. let:- The scope of a let variable is block scope. It can be updated but cannot be re-declared into the scope. It can be declared without initialization. It cant have variables with same name. const:- The scope of a const variable is block scope. It cannot be updated and re-declared into the scope. It cannot be declared without initialization. It can have variables with same name. In execution context we allocate the actual values to the variables instead of the default value. A programing language is said to have First-class functions if functions in that language are treated like other variables.
